### tar 包

下载好后

```bash
tar -xzf nvim(Tab)
sudo mv nvim(Tab) /opt/nvim
sudo ln -s /opt/nvim/bin/nvim /usr/local/bin/nvim

# 测试版本号
nvim --version
```

## appimage

下载好后

```bash
chmod +x nvim(Tab)
sudo mv nvim(Tab) /opt/nvim.appimage
sudo ln -s /opt/nvim.appimage /usr/local/bin/nvim

# 测试版本号
nvim --verison
```
