# 安装 Fcitx5 输入法

```bash
sudo pacman -S fcitx5 fcitx5-configtool fcitx5-chinese-addons fcitx5-gtk fcitx5-qt
```

![](https://img.makis-life.cn/imagesNew/2026-01-29-3.png?x-oss-process=style/yasuo)
打开 fcitx5-configtool 就是配置界面，在右侧搜索 pinyin,双击上面那个，就可以添加拼音输入法，下面是五笔，我也不会用 hh
![](https://img.makis-life.cn/imagesNew/2026-01-29-4.png?x-oss-process=style/yasuo)
在 tab 栏选择`Global Options`第一个就是切换输入法的快捷键，方便切换中英，默认有`Ctrl+Space`,习惯 windows 的可以添加`Win(Super)+Space`

![](https://img.makis-life.cn/imagesNew/2026-01-29-5.png?x-oss-process=style/yasuo)

Tab 栏 Add on 中，有配置输入法提示窗的外貌的。
