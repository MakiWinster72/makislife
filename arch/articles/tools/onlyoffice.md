# 安装 onlyOffice

有人推荐 WPS,但是他的界面实在是有点丑，而且中文字体经常出现黑团的现象。虽然似乎是字体问题，手动安装宋体（simsun）等能修复，但是实在是没想到他居然不会回滚...而且也不知道为什么相比其他 office 软件他安装占用能大那么多，经典国产。

## 安装

需要先安装 AUR 助手 ==> [install_AUR_Assis](../install_AUR_Assis.md)
然后

```bash
yay -S onlyoffice-bin
```
