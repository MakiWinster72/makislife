# 欢迎

这是一个为中文用户整理的 Arch Linux 资源站，目标是把 Arch 的灵活性和强大生态以更易读的方式呈现给使用中文的用户。你会在这里找到安装指南、常用软件的安装与配置、以及个人 dotfiles 示例。

## 在本地预览与构建

- 本地开发预览（热重载）：在项目根目录运行 `npm run docs:dev`
- 生成静态站点：运行 `npm run docs:build`
- 预览构建产物：运行 `npm run docs:preview`
